# Desktop App

Desktop package goals:

- Electron + React + TypeScript + Vite
- Switch-inspired launcher layout
- local profile, theme, accent, and library state
- macOS and Windows packaging

## Scripts

- `npm run dev -w @emusol/desktop`
- `npm run build -w @emusol/desktop`
- `npm run dist -w @emusol/desktop`

## Current scope

Current desktop build already includes:

- Switch-style launcher UI,
- local profile and theme settings,
- ROM import and persistent library,
- built-in local emulation for NES, SNES, GB, GBC, GBA, and Mega Drive,
- pause menu with controls, audio, video, autosave, and save slots,
- auto and manual cover workflow.

The next major step is expanding built-in support to the heavier platforms.
