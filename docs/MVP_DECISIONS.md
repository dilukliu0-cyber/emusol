# Emusol MVP Decisions

This file captures the currently agreed product decisions.

## Version 1 priorities

- Local desktop launcher first.
- Switch-inspired interface.
- macOS build support from the first desktop package.
- Core local platform set:
  - NES
  - SNES
  - GB
  - GBC
  - GBA
  - Sega Mega Drive

## Deferred after version 1

- Online rooms.
- Netplay and anti-desync flow.
- iOS and Android support.

## Future platform expansion

- Nintendo 64
- GameCube
- Nintendo DS
- Nintendo 3DS

## Engineering interpretation

The first serious emulator integration should favor one shared path for the version 1 systems, while keeping adapter boundaries clean enough for harder platforms later.
