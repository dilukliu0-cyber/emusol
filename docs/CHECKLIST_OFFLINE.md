- `Готово` Библиотека игр с поиском, фильтрами и сохранением на диск.

## Next platform wave

- `Готово` Встроенный запуск N64 через EmulatorJS.
- `Готово` Встроенный запуск DS через EmulatorJS / DeSmuME.
- `Готово` Финальная DS-схема: основной экран, второй экран, размер зоны, `fit/stretch`, `detach`.
- `Готово` Нативный запуск `3DS` из библиотеки Emusol через внешний совместимый эмулятор.
- `Частично готово` Runtime-паритет для N64: live `video filter`, `aspect ratio`, `integer scale`; дальше нужен QA на реальных ROM и более глубокая работа по контроллерам.
- `Подготовлено` GameCube в архитектуре и библиотеке.
- `Готово` 3DS в архитектуре, библиотеке, UI-статусах и native-launch path.
- `Блокер на 11 марта 2026` В npm-пакетах EmulatorJS не найден официальный `3DS` core, поэтому встроенный запуск внутри окна еще не открыт.
- `Не готово` Встроенные ядра и реальный запуск для GameCube.
- `Следующий этап` Встроенное ядро, запуск внутри окна и схема двух экранов для 3DS.
